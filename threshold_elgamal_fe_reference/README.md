# Threshold ElGamal Fe

This branch contains the clean Fe implementation for a BN254 threshold ElGamal
election contract.

The reference material, Python oracle, deterministic vectors, Fe test ingot,
benchmarks, generated bytecode, reports, and local validation artifacts live on
the `ai-reference-validation` branch.

## Build

```bash
cd threshold_elgamal_fe_reference
fe build
```
